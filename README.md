# Superstore Profitability Analysis (Shiny App)

Interactive R Shiny dashboard for exploring multi-year profitability
by product subcategory and U.S. state.

## Features
- Reactive filtering by category and state
- Profitability trends and year-over-year change
- State rankings and exportable tables
- Light/dark mode toggle

## Tech Stack
R, Shiny, tidyverse, ggplot2, plotly,
